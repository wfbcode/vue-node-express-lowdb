### 介绍

一个简单的全栈项目：[项目访问地址](http://www.wangfangbo.cn/#/login)

### 技术栈

- 前端：vue 全家桶 + vant + axios
- 后端：node + express + lowdb
- 服务器部署：nginx + pm2
